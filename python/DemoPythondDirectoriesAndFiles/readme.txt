Rename all files in current directory not to have numbers in the name.

If successfully achieved, the reorder of images will show complete order of english alphabet.

1)

Go to demo_shellscript folder and note that there are numbers in the name and view shows random message.

Now run rescript.sh from command line. Note that files having space in name is not taken care.

2)

Go to demo_pythonscript folder and note that there are numbers in the name and view shows random message.

Now run rescript.py from command line. Note that you see A-Z in the folder view when auto reordered the view of icons.
